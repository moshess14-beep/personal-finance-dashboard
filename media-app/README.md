# הספרייה שלי — ספרים, סרטים וסדרות

אפליקציית PWA אישית לשמירה וסינון של המלצות: ספרים, סרטים וסדרות.
האפיון המלא: [docs/media-collection-app-spec.md](../docs/media-collection-app-spec.md)

## מה עובד (MVP)

- **העלאת תמונה של המלצה** — קריאת טקסט מהצילום בדפדפן (Tesseract OCR, עברית+אנגלית), בחירת השורה הרלוונטית וחיפוש.
- **הקלדת שם בלבד** — חיפוש מקביל בספרים ובסרטים/סדרות.
- **זיהוי והעשרה אוטומטיים:**
  - ספרים — Google Books (ללא מפתח).
  - סרטים וסדרות — TMDB בעברית (עם מפתח חינמי, מוזן במסך ההגדרות), כולל **זמינות בפלטפורמות סטרימינג בישראל**; ללא מפתח — ויקיפדיה עברית (מידע חלקי).
- **מסך "האם התכוונת ל…?"** — בחירה בין מועמדים לפני שמירה; תמיד יש מסך אישור עם אפשרות עריכה.
- **ספרייה בשתי עמודות** — ספרים מימין, סרטים וסדרות משמאל.
- **סינון** — סוג (ספר/סרט/סדרה), ז'אנר, ופלטפורמת סטרימינג ("כל מה שזמין בנטפליקס").
- **כרטיס פריט** — סטטוס (רוצה/בתהליך/הושלם/נטשתי), דירוג, הערה אישית, סימון ידני של זמינות.

הנתונים נשמרים כרגע ב-localStorage במכשיר. סנכרון בין מכשירים (Supabase) — השלב הבא.

## פיתוח

```bash
npm install
npm run dev          # פיתוח מקומי
npm run build        # בנייה (נפרסת ל-GitHub Pages תחת /media/)
npm run build:demo   # בניית גרסת הדגמה (חיפוש מדומה + נתוני דוגמה)
node scripts/build-artifact.mjs   # אריזת ההדגמה לקובץ HTML אחד
```

## מפתח TMDB

נרשמים בחינם ב-themoviedb.org, מפיקים API Key, ומדביקים במסך ההגדרות באפליקציה.
המפתח נשמר מקומית בדפדפן בלבד.
